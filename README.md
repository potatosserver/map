# OSM 地圖演示 (OSM Map Demo)

這是一個互動式網頁地圖應用程式，用於展示多種 OpenStreetMap (OSM) 圖磚樣式。使用者可以透過簡單的介面即時切換不同的地圖風格，包括針對特定需求優化的樣式、原始風格以及具備 3D 視圖的選項。

## ✨ 功能亮點

*   **多樣化的地圖樣式**：整合了來自 `OSM` 和 `OpenFreeMap` 的多種地圖樣式，提供豐富的視覺選擇。
*   **即時樣式切換**：使用者可以透過側邊的懸浮選單，無縫切換不同的地圖圖層。
*   **動態載入指示**：在切換地圖樣式或載入圖資時，提供明確的「載入中...」提示，提升使用者體驗。
*   **智慧型樣式預載入**：應用程式啟動時會預先載入所有 Maplibre GL 樣式，加快後續切換速度。
*   **地圖樣式標籤**：每個樣式按鈕旁都有清晰的標籤（如：優化、原始、OSM、3D），幫助使用者快速了解其特性。
*   **3D 視圖**：提供一鍵切換至 3D 透視圖的功能，帶來更具沉浸感的地圖瀏覽體驗。
*   **響應式介面**：地圖和控制選單能適應不同大小的螢幕。
*   **可收合的選單**：樣式切換器選單支援展開與收合，讓使用者在不需要時能有更完整的地圖視野。
*   **在地化文字優先**：地圖圖層經過預處理，會優先顯示繁體中文 (`name:zh-Hant`)、簡體中文 (`name:zh`) 的地名。

## 🚀 如何使用

1.  直接在網頁瀏覽器中開啟[演示網頁](https://potatosserver.github.io/map)。
2.  地圖初始化後，預設載入 "Shortbread" 樣式。
3.  點擊畫面右上角的「地圖樣式」選單來查看所有可用的樣式。
4.  點擊任一樣式按鈕（例如 "Colorful", "OpenStreetMap", "3D"）即可切換至該地圖風格。
5.  當選擇 "3D" 樣式時，地圖會自動傾斜並拉近，以展示 3D 效果。切換回其他 2D 樣式時，視角會自動恢復為平面。
6.  點擊「地圖樣式」標題列可以收合或展開選單。

## 🛠️ 技術棧

本專案使用了以下前端技術與函式庫：

*   **[Leaflet](https://leafletjs.com/) (v1.9.4)**：一個輕量、高效的開源 JavaScript 互動式地圖函式庫，作為本地圖應用程式的底層框架。
*   **[MapLibre GL JS](https://maplibre.org/) (v3.6.2)**：用於渲染向量圖磚的高效能開源函式庫，是實現多樣化地圖風格的核心。
*   **[@maplibre/maplibre-gl-leaflet](https://github.com/maplibre/maplibre-gl-leaflet)**：一個將 MapLibre GL 作為圖層整合進 Leaflet 的外掛程式。
*   **HTML5 & CSS3**：用於建構頁面結構與設計樣式，包括使用 Flexbox 進行佈局和 CSS Transitions/Animations 實現流暢的視覺效果。
*   **JavaScript (ES6+)**：處理地圖邏輯、使用者互動、非同步樣式載入 (`async/await`) 等。
*   **[Google Fonts (Material Symbols)](https://fonts.google.com/icons)**：用於介面中的圖示（如下拉箭頭）。

## 🗺️ 可用地圖樣式

### 即時更新
*   **Shortbread** (`優化`): 經過優化的地圖樣式。
*   **Colorful** (`原始`): 原始色彩風格的地圖。
*   **OpenStreetMap** (`OSM`): 標準的 OpenStreetMap 點陣圖磚樣式。

### OpenFreeMap
*   **Positron**: 一種乾淨、簡約的淺色地圖樣式。
*   **Bright**: 一種色彩鮮明的地圖樣式。
*   **Liberty**: 具備獨特風格的地圖樣式。
*   **3D** (`3D`): 以 Liberty 樣式為基礎，並提供 3D 建築物視圖。
